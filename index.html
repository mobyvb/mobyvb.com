Bot Wars

Bot Wars

This was a web app I built working at Skookum. It was intended as a challenge for developers at Skookum, but the tournament included people from outside the company.

Each participant builds a web server for their bot. During a game, each bot receives a game state and responds with its moves for every turn until one bot is victorious. More detailed rules can be found in the Github repository.

Check it out! Github repo.

Story Game

Story Game

This is a webapp I built based on a game I was very fond of in my freshman year of high school.

The game is played on a piece of paper. Each player reads the previous sentence of a story, and continues the story on the next line. When he/she is done, he/she folds the paper over so only the sentence just written can be read by the next player.

Play the game! Github repo.
